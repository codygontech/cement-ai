## ü§ù Contributing to Documentation

Found an error? Want to improve documentation?

### Repository Branch Structure

The project uses **separate branches** for different components:

- **`docs`** - Documentation (this site)
- **`frontend`** - Next.js application  
- **`backend`** - FastAPI backend

### How to Contribute

1. Fork the repository
2. Clone and checkout the `docs` branch:
   ```bash
   git clone https://github.com/codygontech/cement-ai.git
   git checkout docs
   ```
3. Make your changes to the Markdown files
4. Test locally with Docsify:
   ```bash
   docsify serve .
   # Visit: http://localhost:3000
   ```
5. Submit a pull request to the `docs` branch

See [Contributing Guide](./development/contributing.md) for details.
