# Cement AI Frontend

## Setup Instructions

### Prerequisites
- Node.js 18+
- npm, yarn, or pnpm

### Installation

```powershell
# Install dependencies
npm install
```

### Development

```powershell
# Start development server
npm run dev
```

The app will be available at `http://localhost:3000`

### Build

```powershell
# Create production build
npm run build

# Start production server
npm start
```

### Environment Variables

Create a `.env.local` file if you need to configure API endpoints:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000
```
